# MarkDown - Latex符号

参考：https://blog.csdn.net/m0_37167788/article/details/78603307

## 一般箭头

↑	\uparrow
⇑	\Uparrow
↓	\downarrow
⇓	\Downarrow
←	\leftarrow
⇐	\Leftarrow
→	\rightarrow
⇒	\Rightarrow
↕	\updownarrow
⇕	\Updownarrow
↔	\leftrightarrow
⇔	\Leftrightarrow

## 长箭头

**在上述表示前加一个long/Long即可**

⟵ 	\longleftarrow
⟸	\Longleftarrow
⟶	\longrightarrow
⟹	\Longrightarrow
⟷	\longleftrightarrow
⟺	\Longleftrightarrow
